R2D2.color.color Module

Provides R2D2 default blind-oriented color set. Set can be use for plotting.

Provided colors

Name

value

blue

‘#7b85d4’

orange

‘#7b85d4’

green

‘#83c995’

magenta

‘#d7369e’

gray

‘#c4c9d8’

ash

‘#859795’

yellow

‘#e9d943’

brown

‘#ad5d50’

Example

import R2D2
import matplotlib.pyplot as plt

plt.plot(x,y,color=R2D2.color.blue)

Notes

This table is obtained from https://github.com/matplotlib/matplotlib/issues/9460
