R2D2_py

Introduction

R2D2で実施した計算を取り扱うためのPythonモジュールを公開している。

https://github.com/hottahd/R2D2_py

以下でダウンロードできる。

git clone https://github.com/hottahd/R2D2_py

PythonでR2D2で定義された関数を使うには

import R2D2

とする。すべてのAPIは API Reference に示されている。

R2D2.Data

R2D2には R2D2.Data クラスが定義してあり、これをオブジェクト指向的に用いてデータを取り扱う。

import R2D2
datadir = '../run/d002/data'
d = R2D2.Data(datadir)

などとしてインスタンスを生成する。

help(R2D2)
help(R2D2.Data)

などとすると実行環境で、モジュール全体や各関数の簡単な説明を見ることができる。

R2D2.Data classは R2D2.Read と R2D2.Sync の2つのクラスを以下のようにコンポジットしている。

class Data:
    (some codes...)

    def __init__(self, datadir):
        self.read = Read(datadir)
        self.sync = Sync(datadir)

        (some codes...)

つまり、 R2D2.Data クラスは、 R2D2.Read クラスと R2D2.Sync クラスのメソッドをすべて持っており、それぞれ R2D2.Data.read と R2D2.Data.sync でアクセスできる。

一方、 R2D2.Read の属性(attribute)は、 R2D2.Data の属性としてもアクセスできる。 R2D2.Read の属性は頻繁に使うので、このような仕様としている。一方、 R2D2.Read のメソッドは、 R2D2.Data のメソッドとしてはアクセスできない。
