eos

R2D2.util.eos(data, ro, se, var)[source]

Returns the Table equation of state.

Parameters:

  • data (R2D2.Data, or, R2D2.Read) – Instance of R2D2.Data or R2D2.Read classes

  • ro (float) – Density

  • se (float)) – Entropy

  • var (str) – Variable name, pr, te, en, op

Returns:

qq – Corresponding variable

Return type:

float

Warning

This method is very slow for large numpy array. Please consider using fortraun code.
