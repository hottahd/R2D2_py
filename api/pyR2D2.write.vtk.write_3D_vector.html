write_3D_vector

pyR2D2.write.vtk.write_3D_vector(qx, qy, qz, x, y, z, file, name)[source]

Outputs the 3D vector data in VTK format especially for Paraview

Parameters:

  • qx (numpy.ndarray,float) – x-component vector size of (ix,jx,kx)

  • qy (numpy.ndarray, float) – y-component vector size of (ix,jx,kx)

  • qz (numpy.ndarray, float) – z-component vector size of (ix,jx,kx)

  • x (numpy.ndarray, float) – x coordinate size of (ix)

  • y (numpy.ndarray, float) – y coordinate size of (jx)

  • z (numpy.ndarray, float) – z coordinate size of (kx)

  • file (str) – File name for output

  • name (str) – Name of the variable
